# 🎓 FORMATION E-LEARNING PROMPT ENGINEERING
## **Version Fondamentaux - 3 Heures de Microlearning Interactif**
### Format 100% en ligne pour le grand public

---

## 📱 **ARCHITECTURE MICROLEARNING**

### **Format Global**
- **Durée totale** : 3 heures découpées en 36 capsules
- **Capsules** : 5 minutes maximum chacune
- **Principe** : 1 capsule = 1 concept = 1 pratique
- **Rythme suggéré** : 2-3 capsules par jour (formation en 2 semaines)
- **Accessibilité** : Mobile-first, consultable partout

### **Structure de Chaque Capsule**
```
1. Hook (15 sec) : Accroche visuelle
2. Concept (90 sec) : Explication claire
3. Démo (90 sec) : Exemple concret
4. Pratique (2 min) : Exercice immédiat
5. Recap (15 sec) : Point clé à retenir
```
